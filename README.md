# IPL 2025 Python EDA Project

## Overview
This project presents an **Exploratory Data Analysis (EDA)** of the **IPL 2025 season** using Python.  
The main goal of this project is to analyze match, team, and player performance data to uncover meaningful insights, trends, and patterns from the tournament.

Through this analysis, I worked on data cleaning, transformation, visualization, and interpretation of cricket-related data using Python libraries.

## Objective
The objectives of this project are:

- To perform data cleaning and preprocessing on IPL 2025 data
- To explore team-wise and player-wise performance
- To identify batting and bowling trends
- To analyze match outcomes and winning patterns
- To create meaningful visualizations for better understanding of the dataset
- To improve practical skills in Python, pandas, matplotlib, and seaborn

## Dataset Information
The dataset used in this project contains IPL 2025 match-related information such as:

- Match details
- Teams
- Players
- Venue
- Toss result
- Match winner
- Runs scored
- Wickets taken
- Batting performance
- Bowling performance

## Tools and Technologies Used

- **Python**
- **Jupyter Notebook**
- **Pandas** – for data cleaning and analysis
- **NumPy** – for numerical operations
- **Matplotlib** – for data visualization
- **Seaborn** – for advanced charts and plots

## Key Insights
Some important insights that can be generated from this project include:

- *Top-Batsman*- Sai Sudharshan and Shubhman Gill became Highest Run Scrorer Batsman, scores above the 600 runs - Both played for Gujrat.
- *Top-Chase-Batsman*- Abhishek Sharma emerged as the standout chase batsman, playing a breathtaking knock of 141 runs in just 50 balls in a single innings.
- *Highest Strike-Rate*- Vaibhav Suryavanshi recorded the highest strike rate in IPL 2025, with an impressive strike rate of 252.
- *Highest Wicket-Taker*- Prasidh emerged as the highest wicket-taker of the season, claiming 21 wickets.
